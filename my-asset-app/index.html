📥 CSV一括取込

moomoo・野村證券からエクスポートしたCSVをドロップするだけで自動取込します。既存の同名銘柄は上書き更新されます。

🇯🇵 moomoo 日本株

moomooアプリ → 保有 → 日本株 → CSVエクスポート
列: コード, 銘柄名, 数量, 現在値, 平均取得価額, 評価額

🇺🇸 moomoo 米国株

moomooアプリ → 保有 → 米国株 → CSVエクスポート
列: コード, 銘柄名, 数量, 現在値, 平均取得価額, 評価額（USD）

🏦 moomoo 投資信託

moomooアプリ → 保有 → 投資信託 → CSVエクスポート
列: 銘柄名, 評価額, 保有口数, 評価損益

🏛 野村證券 一括

野村証券 → 保有残高 → CSV（預り資産）ダウンロード
株式・投信・外株をまとめて取込します
